# Main Page QA Checklist

## 구조
- [ ] 섹션 순서가 기준과 동일하다.
- [ ] 메뉴명이 `슬쩍 얘기하기 / 슬쩍 공방 쉐어하기 / 슬쩍 물건 판매하기 / 슬쩍 정보 공유하기`로 표시된다.

## 기능
- [ ] 배너는 이미지 전용이다.
- [ ] 배너 이미지 클릭 시 이동 링크가 동작한다.
- [ ] 배너 자동 롤링이 동작한다.
- [ ] 배너 좌/우 버튼, 도트가 동작한다.
- [ ] 커뮤니티 탭 버튼(`Q&A`, `자랑`, `공유`) 전환 시 목록이 바뀐다.
- [ ] 공방 섹션이 가로 슬라이더로 동작한다.
- [ ] 판매 섹션이 가로 슬라이더로 동작한다.

## 디자인
- [ ] 흰 배경/검은 타이포 중심 톤을 유지한다.
- [ ] 검색창/검색바가 없다.
- [ ] 카드 보더와 라운드 스타일이 유지된다.
- [ ] 와이어프레임처럼 보이지 않고 실제 UI처럼 보인다.

## 반응형
- [ ] Desktop(1440폭)에서 레이아웃이 깨지지 않는다.
- [ ] Mobile(390폭)에서 레이아웃이 깨지지 않는다.
- [ ] 모바일에서 배너/탭/슬라이더 조작이 가능하다.

## 빌드
- [ ] `npm run build` 성공
