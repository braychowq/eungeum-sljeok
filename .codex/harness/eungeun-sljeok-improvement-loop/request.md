# 은금슬쩍 전면 개선 에이전트팀 요청

## 요청 요약
- 은금슬쩍을 전면 개선하기 위한 approval-gated 에이전트팀을 구성한다.
- 역할은 `팀장(planner)`, `서비스 시나리오 기획`, `UX/디자인`, `개발`, `검수`로 나눈다.
- 반복적으로 개선점을 분석하고 도출한 뒤, 안전한 범위에서는 실제 작업까지 수행하는 루프를 만든다.

## 운영 목표
- 메인, 슬쩍 커뮤니티, 공방 쉐어를 같은 서비스군으로 느껴지게 만든다.
- 각 사이클마다 개선 후보를 누적하고, 거시적 프론트엔드/UX 개선은 자동 반영한다.
- 고위험 또는 승인 필요 작업은 backlog로 분리한다.
- 비네트워크 실패는 원인 분석과 제한된 재시도를 통해 같은 사이클 안에서 최대한 복구한다.

## 기본 제약
- repo-local harness 규약이 없으므로 fallback artifact contract를 사용한다.
- 현재 검증 수단의 중심은 `cd frontend && npm run build` 이다.
- 반복 실행 중 사용자 작업을 덮어쓰지 않도록 dirty worktree에서는 자동 구현을 보수적으로 제한한다.
