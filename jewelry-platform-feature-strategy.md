# 은근슬쩍(Silver-ly) 커뮤니티 중심 MVP 단계별 기능명세서

문서 버전: v2.0  
작성일: 2026-03-03  
적용 방향: 커뮤니티 중심 MVP

## 1) 제품 목표

은근슬쩍의 초기 MVP는 `공간 공유`, `자원 순환`, `정보 연결` 3가치를 중심으로, 제작자들이 매일 들어오는 커뮤니티 허브를 만드는 데 집중한다.

초기 성공 기준:
1. 제작자 간 질문/답변과 작품 공유가 활발히 일어남
2. 공방 예약 요청과 소규모 물품 거래가 실제로 발생함
3. 운영자가 최소 인력으로 콘텐츠/거래 신뢰를 관리할 수 있음

## 2) 단계별 릴리즈 계획

## 3) 사용자 권한 원칙

역할 기반 회원 분리는 적용하지 않는다. 모든 사용자는 동일한 회원 유형으로 가입한다.  
기능 접근은 역할이 아니라 `인증 상태`와 `콘텐츠 소유권` 기준으로 부여한다.

1. 가입 즉시 가능: 커뮤니티 이용, 검색, 예약 요청, 거래 조회
2. 본인 인증 후 가능: 공방 등록, 상품 등록
3. 소유 리소스에 한해 가능: 예약 승인/반려, 판매 상태 변경
4. 운영 계정 전용: 신고 처리, 블라인드, 카테고리/공지 관리

## 5) 단계별 완료 기준(DoD)

| 단계 | 완료 기준 |
|---|---|
| 단계 1 | 일일 게시글/댓글 생성, 신고 처리, 검색 동작이 안정적으로 운영됨 |
| 단계 2 | 공방 예약 요청-응답 흐름과 마켓 등록-거래후기 흐름이 실제 사용자로 검증됨 |
| 단계 3 | 거래처/지원사업 콘텐츠 정기 업데이트와 운영 대시보드 지표 확인이 가능함 |

## 6) KPI

1. 커뮤니티: DAU, 게시글 작성률, 댓글 참여율, 7일 재방문율
2. 공방 쉐어: 예약 요청 수, 승인율, 예약 성사율
3. 마켓: 신규 등록 수, 거래 완료율, 후기 작성률
4. 정보 창고: 콘텐츠 조회수, 북마크율
5. 운영: 신고 처리 시간, 정책 위반 재발률
